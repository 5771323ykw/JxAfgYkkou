# Vue3商品交易系统 java429

## 项目概述

本项目是一款基于Vue3的商品交易系统，主要包括商城后台和用户管理功能。整个系统采用纯前端实现，无后台及数据库支持，是一个完整的Vue3项目作业。

## 技术栈

本项目采用以下技术搭建：

- **项目框架：** `Vue3` + `Vite`
- **路由管理：** `Vue Router 4+`
- **数据请求：** `Axios`
- **UI实现：** `Element Plus`
- **编程范式：** 使用Vue3的`setup`语法糖

## 功能模块

- **用户管理：** 实现用户的登录和注册功能
- **购物车：** 实现商品添加、删除以及查看购物车功能
- **后台管理：** 实现商品管理等功能

## 系统角色

- **普通用户：** 可进行注册、登录、添加商品到购物车等操作
- **管理员：** 暂无实际角色划分，但预留后台管理功能

## 运行环境

- **推荐环境：** 现代浏览器（Chrome、Firefox等）
- **依赖要求：** Node.js版本需符合Vue3和Vite的最低要求

## 部署步骤

1. 克隆项目代码到本地
2. 安装项目依赖（使用npm或yarn）
3. 运行项目（开发环境：npm run dev 或 yarn dev）

## 核心亮点

- 利用Vue3的`setup`语法糖，简化代码结构，提升开发效率
- 使用Vite作为构建工具，实现快速开发及热加载
- 使用模拟数据，实现前后端分离，便于后续与真实后端服务对接
